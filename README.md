### ⚠️ Prerequisite: Extraction
To ensure the application runs correctly and no files are lost, follow these steps:
1. **Download WinRAR:** [Official Download Page](https://www.win-rar.com/download.html?&L=0)
2. **Extract Files:** Move the contents of the `.zip` or `.rar` into a **new folder** on your desktop. *Do not run the files from inside the compressed folder!*

---

### 📖 Usage Instructions

1. **Launch the App** Right-click the `.exe` file and select **Run as Administrator**.

2. **Capture the File Path** Locate the file you want to use, **Right-Click** it, and select **"Copy as path"**.
   > *Example output: `"C:/Users/Desktop/my-project.file"`*

3. **Configure the Path** * Delete the default text inside the application's path box.
   * Paste your copied path.
   * **Important:** Delete the `"` double quotes from the start and end of the text you just pasted.

4. **Execution** * Open **Notepad**.
   * Press **Run** in the application.
   * *Note: If it fails to run, ensure your antivirus or specific game-security software is temporarily disabled.*
