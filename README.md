## 📖 Step-by-Step Usage Guide

<details>
<summary><b>Click to expand detailed instructions</b></summary>

### Step 1: Administrator Privileges
Right-click the project `.exe` and select **Run as Administrator**. This ensures the application has the necessary permissions to interface with your files.

### Step 2: Copying the Target Path
Navigate to the file you wish to utilize. 
* Hold **Shift** + **Right-Click** on the file.
* Select **"Copy as path"**.
* *Visual Check:* Your clipboard now contains something like `"C:\Users\Name\Desktop\target.file"`.

### Step 3: Application Configuration
1. Go to the application window.
2. Locate the **File Path Box**.
3. Highlight and **Delete** any existing default text.
4. **Paste (Ctrl+V)** your copied path.
5. **Crucial:** Manually delete the `"` (quotation marks) from the very beginning and the very end of the text.

### Step 4: Final Execution
1. Open a blank **Notepad** window.
2. Return to the application and press **Run**.
3. **Troubleshooting:** If the application fails to launch, ensure your security software/firewall is not blocking the process.
</details>
